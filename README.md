# Biodata Website - Setup Instructions

## Files Created:
1. `index.html` - Main HTML structure
2. `styles.css` - Styling and animations
3. `script.js` - Interactive functionality
4. `README.md` - This instruction file

## How to Customize Your Biodata:

### 1. Replace Placeholder Information in `index.html`:
- `[Your Full Name]` - Replace with your actual name
- `[Your Profession]` - Your job title/profession
- `[Your City, State]` - Your location
- `[Your Age]` - Your age
- `[Your Height]` - Your height
- `[Your DOB]` - Date of birth
- `[Your Religion]` - Your religion
- `[Your Caste]` - Your caste/community
- `[Your Language]` - Your mother tongue
- `[Degree Name]` - Your educational qualifications
- `[University/College Name]` - Institution names
- `[Year of Graduation]` - Graduation years
- `[Job Title]` - Current job position
- `[Company Name]` - Current company
- `[Your Income]` - Annual income
- `[Father's Name]` - Father's name and occupation
- `[Mother's Name]` - Mother's name and occupation
- `[Number & Details]` - Sibling information
- `[Joint/Nuclear]` - Family type
- `[Your Mobile]` - Contact number
- `[Your Email]` - Email address
- `[Your Address]` - Full address

### 2. Add Your Photo:
- Save your photo as `your-photo.jpg` in the same folder
- Recommended size: 400x500 pixels for best fit
- Make sure it's a clear, professional photo

### 3. Color Customization (Optional):
You can change the color scheme by modifying these CSS variables in `styles.css`:
- `#8B4513` and `#A0522D` - Brown theme colors
- `#DAA520` - Gold accent color
- `#667eea` and `#764ba2` - Background gradient

- Test on different devices before sharing
